# Hospital System
A robust Command Line Interface (CLI) application built in C++ to manage patient records efficiently. This system allows healthcare administrators to handle patient data, prescriptions, and contact details with automated data persistence.
Features

• Patient Records Management: Add, update, and delete patient profiles.

• Search Functionality: Overloaded search functions to find patients by Name or ID.

• Data Persistence: Automatically loads data from data.txt on startup and saves changes upon exit.

• Sorting: Organize patient lists alphabetically by name or numerically by ID using optimized sorting algorithms.

• Automated Reporting: Generates a report.txt file tracking the total number of records and the timestamp of the last update.

• Detailed Profiles: Tracks Name, ID, Age, Phone Number, Medicine, and Dosage.
